# Homebrew Tap

Homebrew formulae for tools by [@drewburchfield](https://github.com/drewburchfield).

## Installation

```bash
brew tap drewburchfield/tap
```

## Available Formulae

### clings

A feature-rich CLI for Things 3 on macOS.

```bash
brew install clings
```

See [drewburchfield/clings](https://github.com/drewburchfield/clings) for details.
